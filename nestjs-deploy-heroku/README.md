## Description

This a project is exemple to deploy api with CI/CD in github actions and deploy in Heroku

---
## Installation

```bash
$ yarn install
```

## Running the app

```bash

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn run test

```
